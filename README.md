🛒 Amazon Clone (HTML & CSS)

This project is a front-end clone of the Amazon homepage built using HTML and CSS. It replicates the layout and design of Amazon’s user interface to practice web development and styling skills.

🚀 Live Demo
You can view the live project here:
👉 https://burhanuddin-dev.github.io/amazon-clone-html-css/

📌 Features
Responsive Amazon-like homepage layout
Navigation bar with logo, search bar, and menu options
Product sections and grid layout
Footer similar to Amazon
Clean and structured UI using pure CSS

🛠️ Technologies Used
HTML5
CSS3
